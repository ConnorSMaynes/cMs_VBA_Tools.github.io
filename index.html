Welcome to the VBA Tools of Connor S. Maynes (cMs)

Welcome all! I hope you find these functions and processes as helpful as I have. Each of these files will help to solve a common problem when coding in VBA. I think those who are just starting to code in VBA will find these tools particularly helpful. I have added a bunch of comments, but in case those are unclear, please feel free to email me (connormaynes@gmail.com | @ConnorSMaynes) or leave a message on GitHub and I will do what I can to help. Also, if anyone has suggestions for improvements or more things they think should be added, feel free.
